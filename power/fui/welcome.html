Welcome to PowerOS! A fully-featured operating system built with JavaScript, featuring a custom kernel, modern GUI, command-line shell, and multiple built-in applications.

⌨️ Full Shell

Complete command-line interface with built-in commands and utilities

6 applications: Calculator, Browser, Editor, File Manager, Terminal, Settings

🔌 System API

Complete API for applications to access kernel services

🚀 Extensible

Easy to add new applications and kernel features

🚀 Quick Start

Option 1: Launch the GUI

You're already here! Just click the button above to open the PowerOS desktop environment.

Option 2: Use the Command Line

Open a terminal and run the shell server:

npm start

Then use commands like:

help sysinfo ls echo "Hello PowerOS" calc 2 + 2

📚 Available Commands

help Show available commands ls [path] List directory contents cd Change directory mkdir Create directory cat Display file contents echo Print text run Run application ps List running processes kill Terminate process sysinfo System information calc Calculator exit Exit shell

🎮 GUI Applications

Once in the GUI, you can run these applications:

  • 📁 File Manager - Browse files and directories
  • 📝 Text Editor - Create and edit text files
  • 🧮 Calculator - Full calculator with operations
  • 🌐 Browser - Simulated web browser
  • ⌨️ Terminal - In-GUI command interface
  • ⚙️ Settings - System configuration

📁 Project Structure

PowerOS/ ├── kernel/ # Custom kernel implementation ├── fui/ # Graphical user interface ├── shell/ # Command-line shell ├── apps/ # Built-in applications ├── boot/ # System startup ├── server.js # HTTP server └── README.md # Full documentation

💡 Tips

  • Drag windows by their title bar to move them around
  • Click the taskbar buttons to switch between open applications
  • Use the start menu (🔵 PowerOS) to launch new applications
  • Terminal supports standard shell commands
  • Calculator handles basic arithmetic operations
